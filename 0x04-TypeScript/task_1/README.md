# Task 1 - Advanced Interfaces and Classes

## Description

This task explores advanced TypeScript features including readonly properties, optional properties, index signatures, interface extension, and class implementation.

## Learning Objectives

- Use readonly modifiers
- Work with optional properties
- Implement index signatures for dynamic properties
- Extend interfaces
- Create and implement class interfaces
- Define function type interfaces

## Key Concepts

### Teacher Interface
- Readonly `firstName` and `lastName`
- Optional `yearsOfExperience`
- Dynamic properties using index signature

### Directors Interface
- Extends the `Teacher` interface
- Adds `numberOfReports` property

### StudentClass
- Implements a defined interface
- Demonstrates constructor usage
- Shows method implementation
