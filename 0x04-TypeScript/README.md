# TypeScript Learning Project

This project contains a series of TypeScript exercises covering fundamental to advanced TypeScript concepts.

## Project Structure

- **task_0** - Basic TypeScript interfaces and DOM manipulation
- **task_1** - Advanced interfaces, classes, and function types
- **task_2** - Interface implementation and type guards
- **task_3** - Ambient declarations and module imports
- **task_4** - Namespaces and declaration merging
- **task_5** - Nominal typing with branded types

## Prerequisites

- Node.js (v12 or higher)
- npm or yarn

## Setup

Each task folder contains its own configuration. To run a specific task:

```bash
cd task_X
npm install
npm run build
```
